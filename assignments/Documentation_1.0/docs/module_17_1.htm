Integration with Paypal

Integration Php Web Quiz with Paypal

 

To integrate with paypal you need to have business account , not personal .

 

* It will be better if you test Paypal with Sandbox account before

 

1) Open config.php file and enter you paypal account details

 

  define("PAYPAL_ENABLED","yes"); // no/yes enable disable integration with paypal

  define("PAYPAL_SELLER_EMAIL","els11@yandex.ru"); // email of your business paypal account

  define("PAYPAL_NOTIFY_SUCCESS_PAYMENT","yes"); // yes/no receive e-mails on success payment

  define("PAYPAL_NOTIFY_FAIL_PAYMENT","yes"); // yes/no receive e-mails on fail payment

  define("PAYPAL_CURRENCY","USD"); // currency of your paypal account

  define("PAYPAL_DATA_NAME","Load balance"); // Data name during the payment

  define("PAYPAL_USE_SANDBOX","yes"); // yes/no .   yes = sandbox , no = production

 

2) Login into your Paypal business account

 

3) Click My Account -> Profile -> My selling tools

 

 

4) Click Update button on "Website preferences"

 

 

5) Turn on "Auto return"

 

6) Enter the below return url

 

http://YOUR_SITE/index.php?module=my_balance&check=1

 

In our web site , the return URL is : http://phpexamscript.net/phpwebquiz_business/index.php?module=my_balance&check=1

 

7) Click My Account -> My profile -> My selling tools and click "Update button" on Instant Payment Notifications .

 

 

8)   Turn on Notifications and enter Notification URL as below :

 

 

http://YOUR_SITE/modules/payment_methods/paypal/ipn.php

 

In our web site the Notification URL is :

 

http://phpexamscript.net/phpwebquiz_business/modules/payment_methods/paypal/ipn.php

 

 

-----------------------------------------------------------------------------------------------

 

All payments will be storing in payment_orders table
