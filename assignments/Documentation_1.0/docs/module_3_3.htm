Users

By default we have one user , called "sysadmin" . This role has marked as system user and cannot be deleted .

But you can add new users and assign any role to this user .

 

Creating a new user

1) Go to Users->New user

2) Fill information about user .

3) Click "Save" button .

 

 

What happens in backend ?

 

Information about users are storing in "users" table as below .

 

 

All uploaded photos are storing in "user_photo" directory.
