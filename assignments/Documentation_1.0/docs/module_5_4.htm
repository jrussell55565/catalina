Point calculation

By default PHP Web quiz uses "COMPLETE" point calculation mode . But you can change it from "config.php" file to "PARTLY" calculation mode .

 

What is the difference ?

 

We have 4 kind of questions . One answer, Multi answer, Multi text and Free text .

 

For one answer and Free text it's the same . It's different for Multi answer and Multi text based questions .

 

Here is the description of point calculation on examples .

 

1) Radio button based question

For "results by point " - Total point for this question is the point that you entered in "Add question" page.

For "results by percent " - Total point for this question is - 100 / total questions count . In our example we have 4 questions in our Quiz and the total point for this question will be 100/4 = 25 .

 

2) Free text question

This question also will be calculated as previous .

 

3) Check box based question

This question has 2 correct answers . In "COMPLETE" calculation mode, If you select only one correct answer no point will be calculates . But it will calculate penalty point .

 

"Complete mode , results by point   " - Total point will be 0 . You need to answer to both questions .

"Partly mode , results by point " - Total point will be - (questions point / count of correct answers) * count of correct answers checked by user .   In our case it will be   ( 6 / 2 ) * 1 = 3 . No penalty will be calculated .

 

"Complete mode results by percent " - Total point will be 0 . You need to answer to both questions .

"Partly mode , results by percent " - Total point will be - (100/questions count / count of correct answers) * count of correct answers checked by user. In our case it will be   (100/4/2) * 1 = 12.50.

 

4) Multi text based question

 

This question will be calculates as previous .

 

"Complete mode , results by point   " - Total point will be 0 . You need to answer to both questions .

"Partly mode , results by point " - Total point will be - (questions point / count of correct answers) * count of correct answers checked by user .   In our case it will be   ( 4 / 4 ) * 2 = 2 . No penalty will be calculated .

 

"Complete mode results by percent " - Total point will be 0 . You need to answer to all questions .

"Partly mode , results by percent " - Total point will be - (100/questions count / count of correct answers) * count of correct answers checked by user. In our case it will be   (100/4/2) * 1 = 12.50.

 

-------------------------------------------------------------------------------------------------

 

When you select "results by point" and user answered "wrong" , it calculates the value that you entered in "Add question" page as penalty point . But it doesn't calculate any penalty point if user left question empty .

 

This is if you have selected   "Calculate points by : question point " , otherwise entered points in create assignment page will be used
