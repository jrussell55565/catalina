Installing

1) Create database in your MySQL or use existing database . Charset of database should be UTF8

Example : CREATE DATABASE myexamdatabase1 CHARACTER SET utf8 COLLATE utf8_general_ci;

2) Be sure , that you have access for creating Tables , Stored procedures , Views and Triggers

1) Unzip your files into your "WWW" folder .

2) Open http://your_site/elsphpwebquiz_subjects_1.0 and fill the installation form

3) After succesfully installation you will get below screen .

 

 

4) Click "Login page" and you will be redirected to login page.

5) Be sure that , your "install" folder has been deleted . If not , delete it manually

 

-------------------------------------------------------------------------------------------------------

 

What happens when you click installl button ?

 

1) Config_install.php is binding with inserted values and copying to root directory

2) tables_and_data.sql file is executing

3) procedures.sql file is executing

4) triggers.sql file is executing

5) Deleting install folder

 

Note : If you want to change any configuration setting after installation , you need to modify "config.php" file manually .

You can find all possible values inside this file , it's commented .
