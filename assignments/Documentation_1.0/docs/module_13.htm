API For importing questions

Importing questions in XML format .

 

To import questions in XML format , you have to post 3 variables to http://yoursite/import/import_quiz.php

     

      a. login

      b. password

      c. data

 

      Login and password variables should be the login and password of your any admin user. Data should be in XML format . Example of XML file for uploading questions you can find near the documentation in "tools" folder.

 

    We have created sample .NET 4.0 application for parsing excel file and importing questions .

     

       

 

1)Select your Excel file

2)Enter login and password of your quiz page

3)Enter URL of your quiz web site ( in my case it's http://localhost/elsphpwebquiz)

4)Press import button

     

You will find Excel template and .NET application with source codes in your "tools" folder.
