Questions

For creating new Question

 

 

a) Go to Subjects->Create Question

b) Click "Add question"

 

C) Fill the question form .

 

1)Select Subject

2)Select difficulty level

3)Question : Enter the text of question in this field

4)Video file : Select video file if you want to add video to question .

        Video file can be in FLV,MP4,F4V formats . We are using "Flowplayer" to display the videos

        All videos are storing in "video_files" directory.

4)Point : Point that will be calculated for this question if user answers correct.

5)Penalty point : Penalty point that will be calculated for this question if user fails.

6)Header text : Header text that will be displayed at the top of question

7)Footer text : Footer text that will be displayed at the bottom of question

8)Success message : Message that will be displayed if user answers correct.

9)Partly success message : This is using only for multi text and checkbox based answers and only if "PARTLY" calculation mode selected.
