Certificates

All certificates are listed in "certificates_folder" directory.

 

To create a new certificate :

 

1) Open "certificates_folder" directory.

2) Create a new folder , this folder will be displayed as name of certificate in assignments page. For example , call it "certificate2".

 

3) Create index.php file inside certificate2 directory . This file will be used for creating certificate . Also put all your images into this directory.

4) Now you can enter your certificate content into this file in "HTML" format.

 

5) This line of code should be as a first line in every certificate.

 

<?php if(!isset($RUN)) { exit(); } ?>

 

6) The list of variables that you can use in this file .

 

[assignment_name],[start_date],[finish_date],[pass_score],[user_score], [UserName], [Name], [Surname],[email],[url]

 

You can test certificate by opening "Settings->Test certificates" .

 

7) After this steps you will see the new added certificate in certificate list on "create assingment" page . This certificate will be converted to PDF format and users will be able to download it .
