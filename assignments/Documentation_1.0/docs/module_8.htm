Mail configuration

Configuration mail .

 

1) Open config.php file and go to section MAIL_FROM .

2 )Enter the below listed information .

 

MAIL_FROM = Your mail address

MAIL_CHARSET = UTF-8

MAIL_USE_SMTP = YES ( use yes or no)

MAIL_SERVER = SMTP address of your mail server

MAIL_USER_NAME = You mail address . ( The same with MAIL_FROM)

MAIL_PASSWORD = Password of your mail address

 

 

3) Example of already configurated php file .

 

 

define("MAIL_FROM", "phpwebquizgold@mail.ru");

define("MAIL_CHARSET", "UTF-8");

define("MAIL_USE_SMTP", "yes");

define("MAIL_SERVER", "smtp.mail.ru");

define("MAIL_USER_NAME", "phpwebquizgold@mail.ru");

define("MAIL_PASSWORD", "phpwebquiz_gold");

 

 

Can use everybody on localhost. But your hosting server will probably close access to mail.ru, so you have to use the mail address of your domain

 

 

Example of my config.php file

 

 

define("MAIL_FROM", "support@aspnetpower.com");

define("MAIL_CHARSET", "UTF-8");

define("MAIL_USE_SMTP", "yes");

define("MAIL_SERVER", "mail.aspnetpower.com");

define("MAIL_USER_NAME", "support@aspnetpower.com");

define("MAIL_PASSWORD", "mypassword");
