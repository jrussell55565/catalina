Branches

By default you have one branch called "Head office" . You can use just one branch , or add as many as you want . You can rename this branch name to your school name , but cannot delete . System should have at least 1 branch.

 

Adding new branch .

 

1) Go to Settings -> Branches

2) Click "New branch"

3) Enter the branch name and description and click "Save" button .

 

All Information about branches are storing in "branches" table .
