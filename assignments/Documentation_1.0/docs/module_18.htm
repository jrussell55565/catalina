Helpdesk

Helpdesk Management System

 

If users have any questions or problems they can ask it via creating a new ticket .

 

1) Click Helpdesk->Create ticket

2) Fill the form and click "Save"

 

3) Once you click "Save" button , the notification mail will be sent to all users that are marked in "Roles->Access management page"

 

 

4) Page will be redirect to the list of Tickets . Administrators can assign it to technicans . Technicans also can be set in Roles - Access management page .

 

5) After creating tickets administrators and ticket creator can send messages to each other and talk about the issue in "replies" section . To open the "replies" section you need to click subject name .

 

 

 

6) Files also can be attached to the ticket and to replies also . The allowed file extensions can be set in config.php file as below :

 

$HELPDESK_FILE_FORMATS = array("jpg","jpeg","gif","pdf","txt","xls","xlsx","doc","docx","mpeg","mpg","mp4","mp3","tif","tiff","png");

 

------------------------------------------------------------------------------------------------------

 

all tickets are storing in "tickets" table

information about ticket creators are storing in d_txs table

all ticket replies are storing in ticket_replies table

read/unread marks are storing in ticket_views table
